# Folder: src

## Purpose
The src folder contains the interfaces, their styling

---

## Contents
- `auth/` — The interfaces of the account pages, login and register
- `common/` — Interfaces used across multiple features
- `listingCreation/` — The interfaces of the creating a listing, the form, and the feedback
- `profile/` — The interfaces of the profile and its functions
- `tutorlistingview/` — The interfaces of the detailed tutor listing view and the messaging
